# tweeter board 

## technologies
* Node
* MongoDB
* Express

## how to use
* install dependencies
	```
	npm install
	```

* start mongo server
	```
	mongod --dbpath ./srv/mongodb
	```

* start node server
	```
	node server.js
	```
* open browser
	```
	localhost:8888/api/posts
	```

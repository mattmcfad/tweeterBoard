# tweeter board 

## technologies
#### MEAN
* MongoDB
* Express
* Angular
* Node

## how to use
* install dependencies
	```
	npm install
	```

* start mongo server
	```
	mongod --dbpath ./srv/mongodb
	```

* start node server
	```
	node server.js
	```
* open browser

	```
	localhost:8888/api/posts
	```
